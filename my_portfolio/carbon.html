It was the group project in which we were able to create the carbon Calculator Website prototype for the company named Advice. We fetched the PageSpeed API and Lighthouse result so that we can display relevent Data for calculating carbon Emission from the website.

The Solution we made have 2 pages in which first page is for the url of the website that you wanted to test and it has other fields to be filled in the form then when user submits, it forwarded to the result page with the Screenshot of the website with certain details and also the to make it interactive we have used tools,Toggle and Sliders Toggle and Sliders so that user can see the changes and compare with the original data.

We also made the form through which you can store the result page in our database for future refrence and share result page with colleague.
